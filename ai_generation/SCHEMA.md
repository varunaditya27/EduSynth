# Slide Data JSON Schema

{
  "slides": [
    {
      "index": 0,
      "title": "Short heading",
      "points": ["bullet 1", "bullet 2"],
      "narration": "Friendly teacher explanation (~10–15 seconds).",
      "duration": 12.5
    }
  ]
}

## Requirements:
- 5–7 slides total (based on target length)
- 2–4 bullet points per slide (short phrases, not sentences)
- Narration sounds spoken, not formal
- Duration per slide: ~10–15 seconds
- Total durations ≈ target length in seconds (±10%)
- Index always starts at 0
